table에 border radius 적용하는 법

May 26, 2022

CSS
  • border-collapse
  • 해결방법

border-collapse

border-collapse 속성은 테이블의 각 셀의 테두리 사이에 간격을 조절하는 속성으로 테이블에 border-collapse: separate가 설정되면 셀 테두리 사이에 간격이 생긴다.

table {
  border-collapse: separate;
  border-radius: 10px;
  border: 1px solid black;
}

th,
td {
  border: 1px solid black;
  padding: 10px;
}

이 간격을 없에기 위해 border-collapse 속성을 seperate가 아닌 collapse로 설정하게 되면 border radius가 적용되지 않는 문제가 생긴다.

table {
  border-collapse: collapse;
  border-radius: 10px;
  border: 1px solid black;
}

th,
td {
  border: 1px solid black;
  padding: 10px;
}

테이블의 테두리 사이 간격을 없엔 채로 border radius를 적용하고자, border-collapse를 collapse로 설정하는 대신 border-spacing: 0을 적용하면 테이블 바깥쪽 테두리는 border-radius가 적용되어 둥글게 변하지만, 테이블 내부 셀들의 모서리는 border-radius가 적용되지 않아 의도와 다른 모양을 띄게 된다.

table {
  border-collapse: separate;
  border-spacing: 0;
  border-radius: 10px;
  border: 1px solid black;
}

th,
td {
  border: 1px solid black;
  padding: 10px;
}

해결방법

table에 border radius가 적용되어도 테이블 내부의 셀이 여전히 각진 모양을 띄고 있어서 생기는 문제이므로 테이블 모서리에 위치한 th, td들에 border-radius를 적용시켜주어야 한다.

table {
  border-collapse: separate;
  border-spacing: 0;
}

table thead tr th:first-child {
  border-top-left-radius: 10px;
}
table thead tr th:last-child {
  border-top-right-radius: 10px;
}
table tbody tr:last-child td:first-child {
  border-bottom-left-radius: 10px;
}
table tbody tr:last-child td:last-child {
  border-bottom-right-radius: 10px;
}

th,
td {
  border: 1px solid black;
  padding: 10px;
}

어느정도 의도한 모습에 가까워졌지만 border-collapse: separate가 적용되어 테이블 셀 사이의 border들이 겹쳐지지 않고 접합되어 의도한 border width보다 2배 굵게 보인다. 이는 전체 td,th의 border width와 테이블의 테두리에 해당하는 border의 width를 조절해 해결할 수 있다.

table {
  border-collapse: separate;
  border-spacing: 0;
}

table thead tr th:first-child {
  border-top-left-radius: 10px;
  border-left-width: 1px;
}
table thead tr th:last-child {
  border-top-right-radius: 10px;
  border-right-width: 1px;
}
table tbody tr:last-child td:first-child {
  border-bottom-left-radius: 10px;
}
table tbody tr:last-child td:last-child {
  border-bottom-right-radius: 10px;
}

table thead th {
  border-top-width: 1px;
}
table tbody tr:last-child td {
  border-bottom-width: 1px;
}
table tbody tr td:first-child {
  border-left-width: 1px;
}
table tbody tr td:last-child {
  border-right-width: 1px;
}

th,
td {
  border: 0.5px solid black;
  padding: 10px;
}
